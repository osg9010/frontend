BOM(Browser Object Model)

1. window 객체

window 객체는 최상위 객체이며, 브라우저 내의 모든 요소들과 객체를 포함한다.

1) window.open()

2) 타이머(Timer)

window.setTimeout()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 한 번만 실행한다.

window.setInterval()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 반복해서 실행한다.

2. location 객체

브라우저의 주소 표시줄(URL)과 관련된 객체로 주소와 관련된 정보를 가진 객체이다.

location.reload()

페이지를 새로 고침하는 메소드이다.

location.assign(), location.replace()

두 메소드 모두 페이지를 이동시키지만 location.replace() 메소드는 history에 기록되지 않는다.

3. history 객체

브라우저에서 이동한 문서의 내역을 관리하는 객체이다.
